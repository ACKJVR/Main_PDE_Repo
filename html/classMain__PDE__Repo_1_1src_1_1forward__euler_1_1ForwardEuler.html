Detailed Description

A concrete Forward Euler subclass of TimeStepper.

The Forward Euler method uses a simple forward finite difference approach to estimate a value at the next time step. This is done with the approximation {u(t+dt) - u(t)}/dt = RHS which can be rearranged to give u(t+dt) = u(t) + dt*RHS.
