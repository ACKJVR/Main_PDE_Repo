Perform the full sequence of spatial problem steps.

  1. Set boundary values
  2. Log the current time step and function values
  3. Evaluate the approximate RHS values

This method calls the other methods defined in this class. Returns the input grid with updated boundary values and the grid of RHS values that will be passed to the time stepper.
